# Student-Performance-Analysis-Dashboard-Power-BI-
Project Overview
This project focuses on analyzing student academic performance using Power BI to uncover meaningful insights and trends. The dashboard helps visualize key performance indicators such as marks, attendance, subject-wise performance, and overall student outcomes.
The goal of this project is to transform raw student data into interactive and easy-to-understand visual reports that support better academic decision-making.
